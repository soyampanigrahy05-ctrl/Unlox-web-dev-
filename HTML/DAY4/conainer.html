CONTAINER

Container is a class in CSS that is used to create a fixed-width container for content. It is often used to center content on a page and to create a consistent layout across different screen sizes.

Example of Container

Here is an example of how to use the container class in CSS:

            .container {
                width: 80%;
                margin: 0 auto;
            }

This will create a container that is 80% of the width of the viewport and centered on the page.
